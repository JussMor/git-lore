# git-lore status

## Description

Displays the current status of your local Git-Lore workspace.

This command prints:

- Total atoms being tracked.
- Entropy score and contradictions (if any).
- Pending rules and their state.
- Insights into decision-heavy scopes.

## Usage

`git-lore status [PATH]`

- `[PATH]`: Optional root to scan. Defaults to `.`.
