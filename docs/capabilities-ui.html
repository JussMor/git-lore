Comandos separados por objetivo

Listar / observar

git-lore status .test
git-lore context --file src/app.rs .test
git-lore explain --file src/app.rs .test
git-lore validate .test
git-lore mcp .test

Explicar / decidir / aplicar

git-lore propose --file src/app.rs --title "..." .test
git-lore signal --path "src/**" .test
git-lore checkpoint --message "..." .test
git-lore set-state --atom-id <id> --state accepted --reason "..." .test
git-lore commit --message "..." .test
git-lore sync .test

Consejo: si el preset falla por restricciones del navegador, usa "Seleccionar carpeta .test manualmente".
